# CaesarCode
## Постановка задачи
Реализовать алгоритм кодирования Цезаря с помощью сдвига по таблице ASCII, а также его взлом на основе частотного анализа.
## Параметры запуска
    arg[2] - режим работы (Encode/Decode/Hack)
    arg[3] - файл с текстом
    arg[4] - сдвиг
